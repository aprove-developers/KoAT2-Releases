Module LocalSizeBound

module LocalSizeBound: sig .. end

A templated bound is a bound of a certain templated form. The different templates are not disjunctive. The upcoming template set always includes the previous one.

type t

Always smaller or equal to a scaling factor multiplied with the sum of all prevariables and a constant. Examples: x'=x+y , x'=2*(x+y+z) s * (e + sum x1;...;xn)

val mk : ?s:int ->
       ?c:int ->
       ?pos_abs:string list ->
       ?pos_pure:string list ->
       ?neg_abs:string list ->
       ?neg_pure:string list -> [ `Lower | `Upper ] -> t

Creates a templated bound from the template and a string list which represents the variable set.

val is_finite_bound : t -> bool

Returns true if the given local size bound is an actual bound and false, if the local size bound is infinity or -infinity

val equal : t -> t -> bool

Returns if the templated bounds represent the same bound.

val factor : t -> int

Returns the factor of the local sizebound. Raises unbounded, if the local size bound is unbounded

val constant : t -> int

Returns the constant of the local sizebound. Raises unbounded, if the local size bound is unbounded

val vars : t -> VarSet.t

Returns a set of of variables which affect the local sizebound

Returns a set of all variables which monotonical increasingly or monotonical decreasingly affect the local sizebound

val vars_of_sign : [ `Neg | `Pos ] -> t -> VarSet.t

Those are variables with a positive coefficient or a negative coefficient.

val pre_kind : [ `Lower | `Upper ] * [ `Neg | `Pos ] -> [ `Lower | `Upper ]
val to_string : t -> string

Converts the templated bound to a string.

val as_substituted_bound : ([ `Lower | `Upper ] -> Var.t -> Bound.t) -> t -> Bound.t

Takes a function that returns sizebounds for each variable and a local sizebound. Returns a bound representing the local sizebound with each variable substituted in a way that the bound is valid.

val as_bound : t -> Bound.t

Converts the templated bound to an actual bound.

val as_formula : Var.t -> t -> Formulas.Formula.t

Returns a formula which expresses that the variable is smaller or equal to the bound, e.g. x <= b.

Tries to find a templated bound of any of the defined templates.

val sizebound_local : Program.t ->
       [ `Lower | `Upper ] ->
       ProgramTypes.Transition.t -> Var.t -> t Batteries.Option.t

Returns a local sizebound of the specified kind for the variable of the transition. A local sizebound is expressed in relation to the values directly before executing the transition.

       Batteries.Option.t

If for all result variables of the given kind a local sizebound is defined, this function returns a local sizebound function. Otherwise it returns None.
