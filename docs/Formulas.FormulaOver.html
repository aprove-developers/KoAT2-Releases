Functor Formulas.FormulaOver

module FormulaOver: 
functor (C : ConstraintTypes.Constraint) -> ConstraintTypes.Formula 
       with type constr = C.t
        and type atom = C.atom
        and type polynomial = C.polynomial
        and type value = C.value

Constructs a default formula using a list of atoms each comparing two polynomials

C : ConstraintTypes.Constraint

Following methods are convenience methods for the creation of formulas.

val lift : atom -> t

Lifts an atom to a formula, i.e., a single atom a is a formula [[a]] as a formula is a disjunction over constraints.

val mk : constr -> t

Creates a formula from a set of constraints.

val mk_true : t

Creates the trivial formula true, i.e., true.

val mk_and : t ->
       t -> t

Returns the (normalized) conjunction of two formulas.

val mk_or : t ->
       t -> t

Returns the disjunction of two formulas.

val neg : t -> t

Returns the negated formula.

val implies : t ->
       t -> t

Returns for two formulas f,g the the formula f -> g

val constraints : t -> constr list

Returns a list of all constraints occuring in the disjunction.

val le_than_any : polynomial ->
       polynomial list -> t

Returns the formula (p <= p1 || ... || p <= pn) for a polynomial p and a set of polynomials p1,...,pn.

       polynomial list -> t

Returns the formula (p <= p1 && ... && p <= pn) for a polynomial p and a set of polynomials p1,...,pn.

val all : t list -> t

Creates a formula (f1 && ... && fn) of a list of constraints f1,...,fn.

val any : t list -> t

Creates a formula (f1 || ... || fn) of a list of constraints f1,...,fn.

val turn : t -> t

Negates every atom of every constraint of the formula.

Following methods return certain properties of the formula.

val vars : t -> VarSet.t

Returns the set of variables which are active in the formula. A variable is active, if it's value has an effect on the evaluation of the constraint.

val to_string : t -> string

Returns a string representing the constraint.
