Following methods return certain properties of the constraint.

val is_true : t -> bool

Returns if the constraint is a tautology

val (=~=) : t -> t -> bool
val equal : t -> t -> bool

Stable structural equality, but not an actual equality

val compare : t -> t -> int

Stable structural compare, but not an actual compare

val vars : t -> VarSet.t

Returns the set of variables which are active in the constraint. A variable is active, if it's value has an effect on the evaluation of the constraint.

val to_string : ?comp:string -> ?conj:string -> t -> string

Drops all nonlinear atoms from the constraints. Example: (a > 0 && b^2 < 2) gets transformed to (a > 0)
