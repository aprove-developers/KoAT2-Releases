Module MaybeChanged

module MaybeChanged: sig .. end

MaybeChanged monad represents a value that might change during a computation.

A type that is wrapped by the MaybeChanged monad represents a value that might change during a computation. Once it has changed it will be forever marked as changed, but can also be further be manipulated.

type status =
| Changed
| Same

Status of an object.

type 'a t = status * 'a

Type of an object.

val return : 'a -> status * 'a

Returns the monad (Same, o) for an object o.

val (>>=) : status * 'a ->
       ('a -> status * 'b) -> status * 'b

Applies function f on the monad (status, o) and changes status iff status is Same to status of f(o).

val flat_map : ('a -> status * 'b) ->
       status * 'a -> status * 'b

Applies function f on the monad (status, o) and changes status iff status is Same to status of f(o).

val map : ('a -> 'b) -> 'c * 'a -> 'c * 'b

Applies function f on the monad (status, o) and returns the resulting monad (status, f(o)).

val unpack : 'a * 'b -> 'b

Returns the object itself without the monad structure.

val has_changed : status * 'a -> bool

Returns true iff. the object has changed.

val if_changed : ('a -> 'a) -> status * 'a -> status * 'a

Applies function f on the monad (status, o) iff status is Changed and returns the resulting monad (Changed, f(o)) or the original iff status equals Same.

val changed : 'a -> status * 'a

Returns the monad (Changed, o) for an object o.

val same : 'a -> status * 'a

Returns the monad (Same, o) for an object o.
