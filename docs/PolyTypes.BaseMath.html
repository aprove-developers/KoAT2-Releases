Module type PolyTypes.BaseMath

module type BaseMath = sig .. end

Modules including BaseMath define basic math operations. They can be typeclass-like extended by MakeMath.

Modules including BaseMath define basic math operations. They can be typeclass-like extended by MakeMath.

type t

Type of elements.

val zero : t

Returns zero element.

val one : t

Returns one element.

val neg : t -> t

Returns negated element.

val add : t -> t -> t

Returns sum of two element.

val mul : t -> t -> t

Returns product of two element.

val pow : t -> int -> t

Returns element to the power of some provided integer value.
