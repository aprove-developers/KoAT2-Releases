Module Bound

module Bound: sig .. end

Implementation of bounds, i.e., polynomials, exponential terms and max/min terms.

A MinMaxPolynomial is a polynomial which allows the usage of min and max functions

type t 
include PolyTypes.Evaluable
include PolyTypes.Math
include PolyTypes.PartialOrder

Following methods are convenience methods for the creation of bounds.

val of_poly : Polynomials.Polynomial.t -> t

Creates a bound from a polynomial over OutInt.

val of_constant : value -> t

Creates a constant bound from a constant OurInt.

val of_int : int -> t

Creates a constant bound from an integer.

val to_int : t -> int

Transforms a bound into an integer.

val of_var : Var.t -> t

Creates a bound from a Variable.

val of_var_string : string -> t

Creates a bound of a string representing a variable.

val min : t -> t -> t

Returns for two bounds p,q the minimum bound min(p,q).

val max : t -> t -> t

Returns for two bounds p,q the maximum bound max(p,q).

val minimum : t Batteries.Enum.t -> t

Returns a bound representing the minimum of all the values. Raises an exception, if the enum is empty. Use the function infinity for those cases.

val maximum : t Batteries.Enum.t -> t

Returns a bound representing the maximum of all the values. Raises an exception, if the enum is empty. Use the function minus_infinity for those cases.

val infinity : t

Returns the infinity bound.

val minus_infinity : t

Returns the negative infinity bound.

val exp : value -> t -> t

Returns for a positive integer OurInt i and a bound b the new bound b^i.

val abs : t -> t

Returns for a bound b the new (absolute) bound |b| := max(0,b) + max(0,-b)

val max_of_occurring_constants : t -> OurInt.t

Returns for a polynomial bound the maximal occuring constant. TODO doc

val is_infinity : t -> bool

Returns true iff. a bound is infinite.

val is_minus_infinity : t -> bool

Returns true iff. a bound is negative infinty.

val to_string : t -> string

Creates a string representing the bound by calling show with complexity enabled.

val show : ?complexity:bool -> t -> string

Generates a string from a bound and adds the asymptotic complexity if parameter complexity is not assigned to false.

Functions to classify the quality of the bound

Following methods can be used to classify the type of the bound. 

val substitute : Var.t -> replacement:t -> t -> t

Substitutes every occurrence of the variable in the polynomial by the replacement polynomial. Ignores naming equalities.

val substitute_all : t Batteries.Map.Make(Var).t -> t -> t

Substitutes every occurrence of the variables in the polynomial by the corresponding replacement polynomial. Leaves all variables unchanged which are not in the replacement map.

val substitute_f : (Var.t -> t) -> t -> t

Substitutes every occurrence of the variables in the polynomial by the corresponding replacement polynomial.
