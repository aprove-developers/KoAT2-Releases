Module TransitionLabel

module TransitionLabel: sig .. end

Modul handles labels of transitions.

A transition is an edge of a transition system. It connects two locations and is annotated with a guard and an update. A guard has to be fulfiled for a state to reach another state via the transition. An update assigns variables a new value as a linear combination of the old values.
