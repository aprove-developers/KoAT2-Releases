Module Approximation

module Approximation: sig .. end

Implementation of approximations containing time, size and cost-bounds.

Provides default implementations of an approximation

type t 
val empty : int -> int -> t

Returns an empty approximation that does not contain any non-trivial information. That means, that every upper bound is infinite and every lower bound is minus infinite. The first parameter should be the count of transitions in the program. The second parameter should be the count of program variables.

val create : Program.t -> t

Creates an empty approximation of a given program.

val time : t -> ApproximationModules.TransitionApproximation.t

Returns time-approximation.

val size : t -> ApproximationModules.SizeApproximation.t

Returns size-approximation.

val cost : t -> ApproximationModules.TransitionApproximation.t

Returns cost-approximation.

val to_formatted : ?show_initial:bool -> Program.t -> t -> FormattedString.t

Creates a formatted string containing time,size and cost-bounds.

val to_string : Program.t -> t -> string

Creates a string containing time,size and cost-bounds.

val equivalent : t -> t -> bool

Returns true iff. time and size-bounds are equivalent. Costs-bounds are not considered.

Timebound related methods

val timebound : t -> ProgramTypes.Transition.t -> BoundsInst.Bound.t

Returns a timebound for the transition.

val timebound_id : t -> int -> BoundsInst.Bound.t

Returns a timebound for the transition id.

val program_timebound : t -> Program.t -> BoundsInst.Bound.t

Returns a timebound for the program.

val add_timebound : BoundsInst.Bound.t ->
       ProgramTypes.Transition.t -> t -> t

Adds the information that the specified bound is a valid timebound for the given transition. The resulting approximation is guaranteed to be at least as good as the old approximation.

val all_times_bounded : t -> ProgramTypes.Transition.t list -> bool

Returns true iff. all transitions from a given list of transitions are bounded and not infinity.

val is_time_bounded : t -> ProgramTypes.Transition.t -> bool

Returns true iff. a given transition is bounded and not infinity.

Costbound related methods

val costbound : t -> ProgramTypes.Transition.t -> BoundsInst.Bound.t

Returns a costbound for the transition.

val program_costbound : t -> Program.t -> BoundsInst.Bound.t

Returns a costbound for the program.

val add_costbound : BoundsInst.Bound.t ->
       ProgramTypes.Transition.t -> t -> t

Adds a (cost-)bound of a transition to an existing approximation.

Sizebound related methods

val sizebound : t -> ProgramTypes.Transition.t -> Var.t -> BoundsInst.Bound.t

Returns a sizebound for the var of the transition. A sizebound is expressed in relation to the input variable values of the program.

val add_sizebound : BoundsInst.Bound.t ->
       ProgramTypes.Transition.t -> Var.t -> t -> t

Adds the information that the specified bound is a valid sizebound for the given variable of the transition. The resulting approximation is guaranteed to be at least as good as the old approximation.

val add_sizebounds : BoundsInst.Bound.t ->
       ApproximationModules.RV.t list -> t -> t

Add a size bound for all result variables of the list
